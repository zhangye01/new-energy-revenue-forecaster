# 全口径收入预测模型说明

## 产品边界

第一阶段聚焦风电、光伏发电侧项目。项目类型与配储解耦：

- 项目类型：风电、光伏。
- 配储条件：不配储、配建储能。
- 地域维度：全国省级区域。
- 时间维度：选择开始年份，最多预测 30 年。
- 计算粒度：每个预测年度均拆解为 8760 小时。

## 收入分项

| 分项 | 当前公式 | 后续可配置方向 |
| --- | --- | --- |
| 现货市场捕获电价 | 小时上网电量 * 小时现货价 * 捕获系数 | 接入省份真实 8760 现货预测曲线和项目节点捕获系数 |
| 差价机制结算收入 | 小时上网电量 * 差价结算比例 * (机制电价 - 小时捕获价) | 兼容煤电基准、竞价上限、政府授权差价、分年退坡 |
| 中长期合约交易损益 | 小时上网电量 * 合约比例 * (合约价 - 小时捕获价) | 支持年度/月度/分时合约、电量偏差、曲线分解和履约考核 |
| 环境价值兑现 | 小时上网电量 * 环境价值单价 * 兑现比例 | 拆分绿证、绿电溢价、碳资产、补贴 |
| 综合费用 | 小时上网电量 * 综合费用单价 | 拆分市场运营费用、并网运行考核、辅助服务、偏差考核 |
| 其他收入项 | 小时上网电量 * 其他收入单价 | 支持容量补偿、需求响应、租赁、地方奖励 |

## 对齐顺序

1. 先对齐年度上网电量。
2. 再对齐现货捕获电价与现货收入。
3. 再对齐差价机制和中长期交易损益。
4. 再对齐环境价值、综合费用、其他收入。
5. 最后对齐税费、贷款、现金流、IRR、NPV。

这样排查差异最快，因为收入侧和财务侧不会互相遮蔽。
