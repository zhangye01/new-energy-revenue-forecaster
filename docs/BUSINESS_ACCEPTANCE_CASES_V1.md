# 验收用例 V1（业务层）

## 1. 用例说明

- 目标：验证业务口径、流程状态、提示机制是否符合 V1 约定。
- 级别定义：
  - P0：主流程阻断或口径错误。
  - P1：结果可用但存在偏差/风险。
  - P2：可优化项，不阻断交付。

2. 执行 AC-006, AC-008, AC-009, AC-010。  
3. 执行 AC-011, AC-013, AC-014, AC-016, AC-018。  
4. 执行 AC-019。
